A simple, elegant appointment booking system for hairstylists and beauty professionals.

## Features

- **Client Booking Page**: Clean interface for clients to view and book available time slots
- **Owner Dashboard**: Manage availability, view bookings, and control your schedule
- **Automatic Slot Management**: Cancelled appointments automatically re-open slots
- **Mobile Responsive**: Works perfectly on phones, tablets, and desktops
- **No Backend Required**: Uses localStorage for simple deployment (can be upgraded to Firebase/Supabase)

## Demo

**Default Password**: `admin123`

## Usage

### For Clients
1. Visit the booking page
2. Browse available time slots
3. Click a slot to book
4. Fill in your contact information
5. Receive confirmation

### For Owners
1. Click "Owner Login" 
2. Enter password
3. Add your available time slots
4. View and manage bookings
5. Cancel appointments if needed (automatically reopens the slot)

## Tech Stack

- React + Vite
- LocalStorage for data persistence
- CSS3 for styling
- GitHub Pages for hosting
